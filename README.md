# Why use this repo?

When you need to backtest your Freqtrade Strategy you need to download data for your assets (like BTC, ETH) for necessary market (like spot or futures)
This takes very much time.
The downloading process takes weeks to complete for all coins for several years.

To tune up your strategy you need this data first
Waiting for downloading data from exchanges it will take 2 weeks to complete
By using this repo and auto downloader script you can directly start backtesting
It is suitable for automated workflows and its designed for most pupular Freqtrade Strategies

# Run this command for download

enter to your freqtrade strategy root directory

select coins which you want to download
make a backup of user_data/data
this script will create a user_data/data

```sh
wget <https://raw.githubusercontent.com/DigiTuccar/HistoricalDataForTradeBacktest/main/internal-tools/Historical-Data-Downloader-For-Backtest.sh>
```
